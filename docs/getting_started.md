## Installation

=== "`uv`"

    ``` sh
    uv add fplstat # (1)!
    ```
    
    1. If you haven't heard of [`uv`](https://docs.astral.sh/uv/), we highly recommend checking it out!

=== "`pip`"

    ``` sh
    pip install fplstat
    ```

## Usage

``` py
from fplstat import FPLstat


# Initialize FPLstat client
fpl = FPLstat()
```
